# MCLCE-Web
Minecraft: Legacy Console Edition ported to run in WebAssembly.


## WHY DID THIS TAKE SO LONG?
because i wanted to take my time working on other things. such as one of my other projects, union : divided. when i was working on this, i really obsessed over flaws and bugs and spent weeks fixing them, just to find more.
i really didn't like criticism and couldnt handle it during march - april, which is why i really bugged out and stopped working on it for a full month.
now, that it's time to release, i feel nothing but happiness
i really am happy i get to share this port with you guys and help you guys enjoy things like this too
source code WILL be released soon, i just need to polish up things in the code (and this is ognna take like 5 months watch)

## How? How'd you fix the UI?
using a custom made swf loader, called LCE-Flash. shockwave-flash turned into lce-flash. yes i made an entire swf thing for this project to work
